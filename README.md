# Ecommerce Website UI

A responsive **Ecommerce Website UI** built using **HTML, CSS, Bootstrap 5, and JavaScript**.  
This project recreates a modern ecommerce landing page based on a **Figma design**.

---

## 🚀 Features

- Responsive navigation bar
- Expandable search bar
- Hero banner section
- Product showcase layout
- Projects gallery
- Responsive footer
- Mobile-friendly layout using Bootstrap
- Optimized images with lazy loading

---

## 🛠️ Technologies Used

- Bootstrap 5

## 📱 Responsive Design

The layout is responsive across different screen sizes:

- Desktop
- Tablet
- Mobile

Bootstrap's grid system is used to ensure proper responsiveness.

---

## 🔍 Search Bar Functionality

The search icon toggles the search input field using JavaScript.

Example logic:

```javascript
document.getElementById("searchIcon").addEventListener("click", function () {
  document.getElementById("searchBox").classList.toggle("active");
});
```
